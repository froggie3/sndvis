# オーディオビジュアライザー

「変換過程のデータ」や「配列の中身」そのものにフォーカスした音声のビジュアライザを作成します。

このビジュアライザは、FFTのアルゴリズム（クーリー・テューキー法など）そのものを可視化します。FFTは入力配列に対して複数の「ステージ」を経て計算を行いますが、その**計算過程の信号の流れ**を可視化します。

* **次元:** 2次元（ネットワーク図）
* **データ:** 時間領域の生データ配列 + FFTの各ステージの計算中間値
* **マッピング:**
* **X軸:** FFTの計算ステージ（Stage 0 〜 Stage N）
* **Y軸:** 配列のインデックス


* **視覚表現:**
* 左端（入力）から右端（出力）へ向かって、網目状（バタフライ演算回路図）にラインが走ります。
* 各交点（ノード）で足し算・引き算が行われている様子を、ノードの「明滅」やラインの「太さ」で表現します。
* **ポイント:** 低音が入力されると全体が大きく揺れ、高周波ノイズが入ると局所的な網目が細かく点滅するなど、アルゴリズムがデータを「噛み砕いていく」様子が観察できます。
